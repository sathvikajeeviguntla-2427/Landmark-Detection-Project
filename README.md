Landmark Detection using Deep Learning
📌 Project Overview

This project implements a Landmark Detection system using deep learning. The model is trained to recognize and classify different landmarks from images using a dataset containing image IDs and corresponding landmark labels.

The system loads image data, preprocesses it, trains a neural network model, and predicts the landmark category present in the image.

This project demonstrates the application of computer vision and deep learning for image classification tasks.

🚀 Features

Image dataset processing

Landmark label extraction from CSV file

Data preprocessing using NumPy and Pandas

Deep learning model using Keras

Image visualization using Matplotlib

Landmark classification from images
